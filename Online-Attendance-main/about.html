ABOUT THE PROJECT

Why We Built SmartAttend

A modern, secure, and anti-proxy online attendance system designed to replace traditional, time-consuming manual attendance tracking in colleges and universities.

Need of this Project

  • To prevent Proxy attendance
  • To save Time
  • To save papers

Key Advantages

Anti-Proxy System

Utilizes real-time OTP generation that expires in just 15 seconds. This ensures only students physically present in the class can successfully mark their attendance.

Lightning Fast

A class of 100+ students can mark their attendance simultaneously within seconds, saving valuable lecture time for actual teaching.

App-Like Experience (PWA)

No Play Store required. Students and faculties can install SmartAttend directly to their home screens as a Progressive Web App.

Cloud Synced

All data is securely stored in a cloud database. Faculties can generate Excel reports and track records from anywhere, anytime.

Role-Based Access

Dedicated and highly secure dashboards for Teachers and Students with persistent login capabilities.
